## Google Forms Clone 
This is a google form clone created with ReactJs.
#### Features
- Add/Delete Elements
- Duplicate ELements
- Set input required
- Drag elements to reorder
- Multiple element types
    - TextInput
    - TextArea
    - RadioInput
    - NumberInput
    - DateInput
    - TimeInput

#### Usage
```bash
git clone git@github.com:bitcoinkash/google-form-clone.git
cd {PROJECTNAME}
npm install
npm start
```
